# Smart Water Governance System

## Description
IoT-based water monitoring and control system based on Jal Jeevan Mission (55 LPCD).

## Features
- Family-based water allocation
- Flow monitoring using sensor
- Leakage detection
- Manual valve control
- RTC-based daily reset
- LoRa communication
- Cloud dashboard

## Architecture
Node ESP32 → LoRa → Gateway ESP32 → Cloud → Dashboard

## Hardware
- ESP32 DevKit V1
- SX1278 LoRa (Ra-02)
- YF-S201 Flow Sensor
- DS3231 RTC
- Relay + Valve
- Solar + TP4056 + Battery + MT3608
